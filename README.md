### 📗 Running the project for the first time (web)

```bash
📗 Installation

Client
1. $ cd client
2. $ yarn install
3. $ yarn start

Server
1. Change your mySQL database data server/index.js
2. $ cd server
3. $ npm i
4. $ npm rum dev
```
